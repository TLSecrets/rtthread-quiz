# RT-Thread 嵌入式操作系统 - 刷题系统
纯静态 HTML/CSS/JS 轻量化刷题网站，零外部依赖，覆盖 RT-Thread 课程 5 周共 62 道习题。支持 GitHub Pages 一键部署。
## 🚀 部署步骤（GitHub Pages）
1. 创建 GitHub 仓库，上传以下文件到仓库根目录：
   - `index.html`
   - `question_bank.json`
2. 进入仓库 **Settings → Pages**，Source 选择 `main` 分支，点击 Save
3. 等待 1-2 分钟后，访问 `https://你的用户名.github.io/仓库名/` 即可使用
## 📂 项目文件
| 文件 | 说明 |
|------|------|
| `index.html` | 完整前端应用（含内嵌题库备用，外部 JSON 加载失败自动启用） |
| `question_bank.json` | 题库数据（外部加载，方便后续单独替换题库） |

## 🛠 题库格式
如需替换为其他题库，编辑 `question_bank.json`，格式如下：
### 单选题
```json
{
  "id": 1,
  "week": 2,
  "topic": "操作系统基础概念",
  "type": "single_choice",
  "title": "题目文本？",
  "options": [
    {"id": "A", "text": "选项A"},
    {"id": "B", "text": "选项B"},
    {"id": "C", "text": "选项C"},
    {"id": "D", "text": "选项D"}
  ],
  "correctAnswer": "B",
  "analysis": "解析文本",
  "memoryTip": "记忆技巧"
}
```
### 填空题
```json
{
  "id": 10,
  "week": 2,
  "topic": "操作系统基础概念",
  "type": "fill_blank",
  "title": "实时系统分为___系统和___系统。",
  "blankCount": 2,
  "answers": [
    {"accepted": ["硬实时"], "primary": "硬实时"},
    {"accepted": ["软实时"], "primary": "软实时"}
  ],
  "analysis": "解析文本",
  "memoryTip": "记忆技巧口诀"
}
```
> 核心约束：单选题正确选项以 `correctAnswer` 的选项 ID 为准；填空题使用 `accepted` 数组做模糊匹配，不区分大小写。
